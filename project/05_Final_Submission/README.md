## What Each File Is

`vln_acm_overleaf.zip` is the complete Overleaf-ready LaTeX report package.

`vln_task3_task4_methodology_slides.pptx` is the two-slide methodology deck for Task 3 and Task 4.

`vln_task3_task4_methodology_slides_bundle.zip` is the slide bundle.

`task45_results.zip` contains Task 4 and Task 5 result visuals and tables.

`task5_regularization_results.zip` contains the Task 5 controlled-extension comparison artifacts.

## How To Use

Upload `vln_acm_overleaf.zip` directly to Overleaf as a new project.

Use the PowerPoint file for presentation. The result ZIPs contain additional graphs/tables if you want to add extra slides.
